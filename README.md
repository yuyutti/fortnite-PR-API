Node.js v22.13.1
npm v10.4.0

FortniteTorakerへヘッドレスブラウザを自動操作でアクセスし、結果を解析してレスポンスを行います。
ウェブスクレイピングを行ってからレスポンスを返すためレスポンスに時間がかかります。  
また、スクレイピングの善悪や危険性を理解したうえでご使用ください。
